# upmark — frontend

This directory holds the Svelte 4 + Vite + TypeScript frontend for [upmark](../README.md). It is not meant to run standalone — it's mounted by Wails inside a WebView.

See the [root README](../README.md) for the full project overview, build instructions, and the [CONTRIBUTING guide](../CONTRIBUTING.md) for local dev setup.

Quick pointer for what lives where:

- `src/App.svelte` — root component, state orchestration, keyboard shortcuts, command palette
- `src/lib/` — TopBar, Sidebar, Viewer, Editor, CommandPalette, URLPrompt, Settings, ImageLightbox
- `src/lib/enhance.ts` — post-render DOM enhancements (callouts, math, mermaid, code-copy, find, task-lists)
- `src/styles/` — `app.css`, `themes.css` (13 `[data-theme=...]` blocks), `markdown.css`
- `wailsjs/` — generated by `wails build`, gitignored
