# Notes

A short companion file so wikilinks have somewhere to land.

This file exists to demonstrate that `[[notes]]` from [sample](sample.md) resolves and opens here. You can navigate back with Alt-←.

## Why a sibling file

Wikilinks resolve by basename across the open folder. If `notes.md` lives anywhere in the folder you opened, `[[notes]]` finds it.

Nothing important here — just enough to populate the contents sidebar with more than one item.
